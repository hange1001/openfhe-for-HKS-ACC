  P0-1：BRAM 超标 206%（最致命）

  根因：top.cpp 第 32-33 行的静态数组：

  static uint64_t NTTTwiddleFactor[MAX_LIMBS][BU_NUM][RING_DIM];  // 5×32×4096×8 = 5MB
  static uint64_t INTTTwiddleFactor[MAX_LIMBS][BU_NUM][RING_DIM];  // 同上，合计 10MB

  关键发现：OP_INIT 中的加载逻辑是广播——所有 32 个 BU 写入完全相同的数据，BU 维度是冗余的。

  修复方案：消除 BU 维度，从 10MB 降到 320KB

  第一步：top.cpp 第 32-33 行

  // 修改前
  static uint64_t NTTTwiddleFactor[MAX_LIMBS][BU_NUM][RING_DIM];
  static uint64_t INTTTwiddleFactor[MAX_LIMBS][BU_NUM][RING_DIM];

  // 修改后
  static uint64_t NTTTwiddleFactor[MAX_LIMBS][RING_DIM];
  static uint64_t INTTTwiddleFactor[MAX_LIMBS][RING_DIM];

  第二步：top.cpp OP_INIT 加载循环（第 106-125 行）

  P0-3：BConv 脉动阵列 30 个硬件除法器

  根因：bconv.cpp 第 114-115 行用 % 做 128 位取模，HLS 综合出 30 个 urem 硬件除法器。

  修复：用已有的 MultMod（Barrett 约减）替换 %

  需要为每个输出模数传入 Barrett 常数 k_half 和 m。

  第一步：bconv.h 更新 Compute_BConv 接口

  void Compute_BConv(
      uint64_t in_x[MAX_LIMBS][SQRT][SQRT],
      const uint64_t in_w[LIMB_Q][MAX_OUT_COLS],
      const uint64_t out_mod[MAX_OUT_COLS],
      const uint64_t out_k_half[MAX_OUT_COLS],   // 新增
      const uint64_t out_m_barrett[MAX_OUT_COLS], // 新增
      int sizeP
  );

  第二步：bconv.cpp 更新 bconv_systolic 和 Compute_BConv

  bconv_systolic 增加参数，PE 计算部分替换：

  // 修改前
  ap_uint<128> prod = ((ap_uint<128>)x_in * (ap_uint<128>)in_w[q][p]) % mod_p;
  ap_uint<128> sum_out = (sum_in + prod) % mod_p;

  // 修改后
  uint64_t prod_64;
  MultMod((uint64_t)x_in, in_w[q][p], (uint64_t)mod_p, out_m_barrett[p], out_k_half[p], prod_64);
  uint64_t sum_64;
  // sum_in + prod_64 做一次 AddMod
  uint64_t sum_tmp = (uint64_t)sum_in + prod_64;
  if (sum_tmp >= (uint64_t)mod_p) sum_tmp -= (uint64_t)mod_p;
  sum_reg[p][q + 1] = sum_tmp;

  注意：sum_reg 类型可从 ap_uint<128> 降为 ap_uint<64>，进一步节省寄存器。

  P1-9：init_Q_Loop II=3，AXI burst 失败

  根因：同一循环内对 mem_in1 有三个不同 stride 的访问，HLS 无法合并为 burst。

  修复：top.cpp 第 72-82 行，拆分为三个独立循环

  // 修改前（三个地址在同一循环内）
  init_Q_Loop:
  for (int i = 0; i < LIMB_Q; i++){
      MODULUS[i] = mem_in1[i];
      K_HALF[i]  = mem_in1[LIMB_Q + i];
      M[i]       = mem_in1[LIMB_Q*2 + i];
  }

  // 修改后（三段连续地址，各自独立 burst）
  init_Q_MOD:
  for (int i = 0; i < LIMB_Q; i++){
      #pragma HLS PIPELINE II=1
      MODULUS[i] = mem_in1[i];
  }
  init_Q_KHALF:
  for (int i = 0; i < LIMB_Q; i++){
      #pragma HLS PIPELINE II=1
      K_HALF[i] = mem_in1[LIMB_Q + i];
  }
  init_Q_M:
  for (int i = 0; i < LIMB_Q; i++){
      #pragma HLS PIPELINE II=1
      M[i] = mem_in1[LIMB_Q*2 + i];
  }
